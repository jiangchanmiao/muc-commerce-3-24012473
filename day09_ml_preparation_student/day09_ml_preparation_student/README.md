## 用自己的话回答

- 什么是特征，什么是标签：特征是模型用来学习的输入信息，比如用户的在网月数、是否投诉、性别、消费金额等；标签是我们希望模型预测的结果，这里指 Churn 是否流失。
- 为什么要保留测试集：测试集用来模拟模型没见过的新用户，评估模型的泛化能力。如果只用训练集评估，模型可能只记住了训练数据，遇到新数据就表现很差，这叫过拟合。
- 为什么83%准确率仍可能没有用：如果数据里大部分用户本来就没有流失，模型即使把所有用户都预测为“不流失”，准确率也可能很高，但这样完全找不出来真正会流失的人。在流失预测里，我们更关心能不能把潜在流失用户识别出来，所以只看准确率不够。
